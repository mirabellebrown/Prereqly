# Prereqly

Prereqly is a Next.js prototype for a UCSB academic planning app. The current demo includes:

- dashboard and student overview
- 4-year planner
- degree checklist
- AI advisor chat
- important dates timeline
- financial aid panel

## Getting started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Available scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Deploy

This app is ready to deploy on Vercel as a standard Next.js project.
